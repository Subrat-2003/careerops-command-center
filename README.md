# CareerOps Command Center

CareerOps Command Center is a local-first job-search workflow tool that helps users track applications, manage referral outreach, generate personalized messages, and organize follow-ups.

## MVP Focus

This MVP is designed to be safe, free, and easy to validate publicly.

## What It Does

- Tracks job applications
- Manages referral contacts
- Generates personalized referral messages
- Tracks follow-ups
- Scores job descriptions based on fit
- Tracks resume versions

## What It Does Not Do

- It does not auto-send LinkedIn messages
- It does not scrape LinkedIn or job portals
- It does not auto-apply to jobs
- It does not store passwords or cookies
- It does not use paid AI APIs in the MVP
