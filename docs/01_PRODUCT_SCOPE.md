# CareerOps Command Center - Product Scope

## Goal

CareerOps Command Center is a local-first job-search workflow tool that helps users track applications, manage referral outreach, generate personalized messages, and organize follow-ups.

## MVP Rule

The MVP must be safe, free, and local-first.

## Included Modules

1. Dashboard
2. Job Application Tracker
3. Referral Outreach Copilot
4. Follow-up Engine
5. JD Fit Analyzer
6. Resume Version Manager
7. Feedback Page
8. About Page

## Not Included in MVP

- No LinkedIn auto-send
- No LinkedIn scraping
- No Naukri, Indeed, or LinkedIn scraping
- No auto-apply
- No paid AI APIs
- No browser extension
- No payment system
- No SaaS billing
- No user password storage

## Storage

Version 1 will use browser localStorage.

## Public Validation

The app should allow users to try demo mode without login.

## Main User Flow

1. User adds a job.
2. User adds a referral contact.
3. User generates a personalized message.
4. User copies the message manually.
5. User sends it manually outside the app.
6. User marks the contact as sent.
7. App creates a follow-up reminder.
8. User checks the Follow-ups page daily.
